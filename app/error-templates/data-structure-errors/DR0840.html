Your file contains unrecognisable field headings

You can only send data under agreed field headings in row 1 of your data returns file.

Check your file against this list of agreed short field headings for optional data:

  • CAS - international standard code for chemicals
  • Ref_Period - reference period for sampling
  • Mon_Period - standard name for the date range covered by the data return
  • Txt_Value - text-based qualifiers
  • Smpl_Ref - an optional reference for the sample, useful for grouping
  • Smpl_By - name of who or what took the sample
  • Site_Name - given on the permit for the location of the sampling
  • Comments - free-text comments about the row of data
