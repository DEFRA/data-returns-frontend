Deliver your data files online

Use this service to send your pollution inventory and landfill emissions data to the Environment Agency.

Why only pollution inventory and landfill emissions data?

This service is being developed to allow data be submitted from a range of applications. While it’s being tested, it’s limited to these two types of data only. For all other types of data submission, please use your normal data returns route.

Before you start

  • Your data files must be in CSV format
    What is this?

    CSV (comma separated values) format is widely used to move data between otherwise incompatible applications. In an Excel spreadsheet, for example, you can use ’Save as’ to create a CSV version of your file.

  • All columns, rows and fields must be correctly labelled
    What does this mean?

    This text explains more about the requisite labelling conventions. Download a sample CSV template to see what these look like.
