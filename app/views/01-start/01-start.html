Before you start

You'll need:

  • an email address and password to create or sign in to your account
  • your permit number
  • your data file (in CSV format only)
Why only CSV format?

CSV (Comma Separated Values) Format is widely used to move data between otherwise incompatible applications

How to structure your CSV file

The file must comprise of:

  • a header row containing the field names
  • the fields sequenced in the field number order indicated
