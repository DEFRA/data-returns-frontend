Send landfill data returns

Use this service to send data returns to the Environment Agency (England only) about:

  • landfill emissions

Report schedule 5 and 6 events directly to your local regulatory team or allocated Site Inspector.

Before you start

Read guidance on how to format your data to meet Environment Agency (EA) requirements.

To use this service you must:

  • save your data as a CSV file
    What's this?

    CSV (comma separated values) format is used to move data between otherwise incompatible applications. Use ‘Save as’ - usually under the File tab - in spreadsheet software to create a CSV version of your file.

  • All columns, rows and fields must be correctly labelled
    What does this mean?

    Download a sample CSV template to see how to label your columns and rows.

    Data returns file structure template

    CSV, 0.5KB

You’ll also need:

  • your permit number
  • an email address to receive a receipt
